csTimer Graphing Tool

Expands on the graphing functionalities found natively in csTimer.

csTimer is an amazing tool for speedcubers, but its options for visualizing time data are limited. This script draws box and violin plots for time distribution, as well as line plots of single and ao12 times against solve number and datetime.

Language: Python
Libraries: pandas, matplotlib
